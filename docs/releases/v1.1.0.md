# portalfin v1.1.0

UX polish + ambient features. Same auth flow, same sideload path as v1.0.0.

## What's new

- **CSS view transitions** — home → library → detail navigations now crossfade in 180ms (ease-out) instead of snapping. The portalfin header gets a `view-transition-name` so it stays put across routes.
- **Splash → home crossfade** — OS splash icon fades + scales (280ms) instead of cutting; the loading-container wordmark animates up + scales down while the WebView crossfades in (240ms).
- **Ambient slideshow at idle** — after 60s of no input, fullscreen rotating gallery of Jellyfin backdrop art with oversized clock + date + current item title. Acquires `FLAG_KEEP_SCREEN_ON` so the Portal display stays lit. Tap to dismiss.
- **Bigger wordmark** — header logo bumped 25% (140×32 → 175×40) and shifted 5px right so the "p" mark lines up cleanly under Portal's back button.
- **Time-of-day theme** — page background subtly shifts: cool morning (#1A1F22), neutral day (#1A1A1A), warm evening (#221A1A), deep night (#0E0E12). 8s ease transition, re-evaluates every minute.
- **Home flash fixed** — the visibility gate that hid the page during initial paint was firing on every SPA route change, causing a flash on every navigation. Now runs once per session.

## Sideload

1. `adb devices` confirms your Portal is connected (with ADB enabled in Settings -> Debug)
2. Download `portalfin-v1.1.0.apk` below
3. `adb install -r portalfin-v1.1.0.apk` (the `-r` upgrades over v1.0.0 in place)

## Known limitations

- Library pages (Movies / TV Shows / etc.) currently show only the back arrow + sort row — the section tabs (Movies / Suggestions / Favorites / Collections / Genres) are temporarily hidden by the custom-header rework. Restoring them is the next planned change.
- The native ExoPlayer chrome is still upstream-default; the default web player is unaffected.
- Portal's UserPresenceManager is aggressive about screen dim; the `FLAG_KEEP_SCREEN_ON` we set during ambient mode helps but may need a Portal-specific override later.
