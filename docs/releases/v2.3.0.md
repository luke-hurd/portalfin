# portalfin v2.3.0

Adds **in-app self-update** — no more manual re-sideloading.

## What's new

### Self-update
- portalfin now **checks for updates on launch** and shows a one-tap prompt when a
  newer release is available.
- A **Check for Updates** button in Profile lets you check on demand.
- Tapping update downloads the latest `portalfin.apk` and launches Android's
  package installer. Installing over the top keeps your login and settings.
- Works precisely because every release ships the same `portalfin.apk` at a stable
  URL, so the app always knows where to get the newest build.

> First update only: Android asks you to allow "install unknown apps" for portalfin
> (a one-time, OS-level security step). After that, updates are a single
> confirmation. This is a normal sideloaded-app updater — not a silent background
> install, which Android doesn't permit.

## Sideload

The download link always points at the newest release:

1. `adb devices` confirms your Portal is connected (ADB enabled in Settings → Debug)
2. Download **[portalfin.apk](https://github.com/luke-hurd/portalfin/releases/latest/download/portalfin.apk)**
3. `adb install -r portalfin.apk` (the `-r` upgrades in place)

After this one sideload, future versions can update themselves from inside the app.
