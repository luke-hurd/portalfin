# portalfin v2.0.0

A ground-up native rebuild. v1.x was a styled shell around the Jellyfin **web**
UI; v2.0 replaces the entire browsing experience with **native Jetpack Compose**
screens that call the Jellyfin REST API directly — *much* faster on the Portal's
modest hardware, and built to Meta's Portal design system (Material 3 + Inter +
Meta blue). The WebView now only backs the few un-ported deep links (person
pages, admin dashboard); everything you touch day-to-day is native.

## What's new

### Native, by design
- **Material 3 + Inter + Meta blue** — the native screens follow Meta's
  portal-samples style guide: M3 components, the Inter typeface (downloadable
  fonts), the `#0866FF` palette, dark-always, 52dp touch targets.
- **Branded launch** — classic windowBackground splash (dark + wordmark) for the
  whole cold start; no circle-masked system icon. New app icon, crisp white logo.

### Native home
- **Home grid** — My Media (your libraries as cards), Continue Watching, Next Up,
  and New Releases per library, as smooth side-scrolling rails.
- **Static header** — centered portalfin logo on a translucent apron that content
  scrolls behind; stays put across screen transitions.
- **Tap feedback + loading shimmer**, animated screen transitions, lazy-loaded
  thumbnails sized for the panel.

### Native library / category pages
- **Paginated** — 100 items per page with Prev / "Page X of Y" / Next controls
  (top and bottom), instead of loading everything at once.
- **Quick-filter pills** — All · Favorites · Genres · Collections.
- **Genres & Collections** render as titled poster grids, each loading lazily.
- 6-column poster grid; library title header.

### Native detail page
- **Fullscreen backdrop** that fades into the page, with content layered on top.
- **Graphical title art** (logo image) when available.
- **Play / Resume / Start Over** (resumes from your saved position).
- **More Like This**, **Cast & Crew**, **Scenes** (chapter picker), **genre chips**.
- **Rotten Tomatoes** critic score, community ★, and an **"Ends at"** estimate.
- **Subtitle picker** (applied on play), tagline, overview.
- Plays through the native ExoPlayer (no WebView).

### Native TV shows
- **Series pages** — "Resume SxxExx / Play S01E01" plus a seasons row.
- **Season detail** — a full detail page (season backdrop, episode grid with
  thumbnail/name/description, and a selector for the other seasons).
- **Episode detail** — the episode still over the series backdrop, with a
  "more from this season" row.

### Native search
- A native search screen backed by the REST API's server-side fuzzy match (the
  same one jellyfin-web uses), debounced as you type, results in the library
  poster grid that open the native detail page. No WebView.

### Native sign-in & settings
- **Connect / server selection / Login** — native Material 3, with the static
  portalfin header and the 64dp Portal top-inset reserve.
- **Profile / settings** — a native screen (no WebView): your account + server,
  Switch Server / Sign Out, and an About section (version, author, license, repo).
- **Search, Downloads and Settings** sit as right-aligned icons on the home
  "My Media" title row (the top OSD band can't take taps, so they live in content).

### Transcoded downloads
- Download any movie/episode — or a whole season — at a chosen quality
  (1080p / 720p / 480p). The server transcodes to a phone-sized file instead of
  the multi-GB original.
- A round download button on the detail page shows download / in-progress /
  downloaded (check + blue ring) states; tapping while downloading or done jumps
  to the Downloads screen.
- The **Downloads screen** lists each item's quality + size and an iOS-style
  device-storage bar (Downloads / Other / Free).
- **Season downloads** check free space first and suggest a lower quality if the
  season won't fit.
- Downloaded copies **play offline and scrub** (MPEG-TS) — Play/Resume uses the
  local file automatically.

### Native video player
- Portal-tuned controls: oversized play/skip icons and scrubber for at-distance
  touch, a larger back button, and the unused buttons (fullscreen, lock, decoder,
  info) removed. Immersive fullscreen now survives the Portal's system-OSD band
  when a control menu opens.

## Sideload

1. `adb devices` confirms your Portal is connected (ADB enabled in Settings → Debug)
2. Download `portalfin-v2.0.0.apk` below
3. `adb install -r portalfin-v2.0.0.apk` (the `-r` upgrades in place)

## Known limitations

- Person/actor pages and the admin dashboard still open the WebView fallback —
  not yet ported to native.
- The top ~64px is the Portal system OSD overlay; interactive UI is kept below it.
- Genre chips on the detail page don't yet dive into that genre.
